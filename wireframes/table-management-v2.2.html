Note di design — v2.2 (cosa cambia rispetto al v2.1)

  • Timeline Gantt rimossa. Duplicava la lista prenotazioni + la mappa sala ed era il pezzo di codice più pesante. TheFork stesso non usa un Gantt: usa lista + mappa + scorri-orari.
  • Colonna "Tavolo" nella lista prenotazioni (Fase 1) — la lista esiste già, si aggiunge solo la colonna. Badge auto per l'assegnazione automatica, "Assegna" rosso se nessun tavolo è stato trovato.
  • Mappa sala operativa (Fase 2) — assorbe il ruolo della timeline: scorri-orari + tavoli colorati per stato; clic su un tavolo occupato → popup "Sposta prenotazione". Doppia modalità: Operativa / Setup.
  • Priorità tavolo: l'ordine della lista in Settings = priorità. L'auto-assegnazione scorre i tavoli in quell'ordine e prende il primo libero con capienza sufficiente.
  • Turni del tavolo: disponibilità per finestra temporale = reservation_time + table_duration + buffer pulizia. Ancorata all'orario prenotato (lo stato "arrivato" non sposta il calcolo).
  • Auto-assegnazione NON bloccante: se non trova un tavolo o va in errore, la prenotazione viene comunque creata senza tavolo. Mai un cancello.
  • Numero tavolo = interno: mai comunicato al cliente che prenota.
  • DB:
    • table_areas: id, tenant_id, name, sort_order
    • tables: id, tenant_id, area_id, name, capacity, priority, shape, position_x, position_y, internal_note, is_active
    • reservation_tables: reservation_id, table_id, is_auto — bridge, multi-riga per le combinazioni
    • table_combinations: tenant_id, table_a_id, table_b_id
    • tenants: table_auto_assign, table_turnover_buffer
    (shape e position_x/y creati subito ma usati solo in Fase 2.)
  • Ordine implementazione:
    1. F1 — migration DB completa
    2. F1 — model Table + Area, CRUD lista per area con riordino priorità
    3. F1 — servizio TableAssigner (turni + priorità + combinazione)
    4. F1 — auto-assegnazione non bloccante alla prenotazione + griglia override nella scheda
    5. F1 — colonna Tavolo nella lista prenotazioni + impostazioni auto-assegnazione
    6. F2 — mappa sala: setup (forme/decorazioni/drag&drop) + operativa (scorri-orari + popup riassegnazione)
